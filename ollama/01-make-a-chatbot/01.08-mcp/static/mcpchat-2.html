Hello! I'm an MCP-powered chatbot using Ollama's native tool-calling API. I can call external tools connected via the Model Context Protocol — for example, try asking "What time is it?" to see the get_system_time tool in action. Unlike the RAG version, I don't have a knowledge base; instead I rely on the model's own knowledge and MCP tools.
