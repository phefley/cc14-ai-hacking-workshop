# ai-hacking-workshop
This is a workshop repo which contains markdown lab documentation and code templates.

During the course, we'll cover three sections:
1. Making a Chatbot
2. Hacking the Chatbot
3. Defending the Chatbot

The examples we've prepared use [Ollama](https://ollama.com/) as the LLM support to keep traffic local and free for workshop participants.

## Setup

You can find instructions for getting setup for the workshop [here](./ollama/setup.md).
